# ddu-source-window

window source for ddu.vim

go to selected window

# Required

## denops.vim

https://github.com/vim-denops/denops.vim

## ddu.vim

https://github.com/Shougo/ddu.vim

## tabby.nvim (Optional)

https://github.com/nanozuki/tabby.nvim

# Example

```
call ddu#start(#{ sources: [#{ name: 'window' }] })
```
